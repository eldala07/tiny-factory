# Tiny Factory Builder

A small browser-based incremental factory game built with Vite, React, TypeScript, and plain CSS.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite.

## Build

```bash
npm run build
```

## Gameplay

Place a miner, run conveyors from the miner's right side, and route ore into a seller. Ore sold earns coins, and coins buy more buildings. Press `R` or use the rotate button before placing conveyors. Progress is saved automatically with `localStorage`.
